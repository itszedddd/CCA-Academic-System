# Project Documentation: House of Puppies (HOP)
**AI-Integrated Management System with SMS Reminder**

This documentation is based on the **[HOP] Thesis Paper** and the **blackjax-grooming** system.

---

### 1. Software Applications Used and Versions
The following software tools and frameworks are utilized in the development and operation of the HOP system.

| Application | Version / Role |
| :--- | :--- |
| **Operating System** | Windows 11 (Minimum) |
| **Development IDE** | Visual Studio Code (Current Stable) |
| **Runtime Environment** | Node.js (v20.x or higher) |
| **Frontend Framework** | Next.js v16.1.5 (React 19) |
| **Backend / ORM** | Prisma v6.4.1 |
| **Database** | SQLite (Dev) / SQL Server (Production) |
| **Styling Framework** | TailwindCSS v4 |
| **Authentication** | NextAuth.js v5.0.0-beta |
| **UI Components** | Radix UI, Shadcn UI |
| **Icons** | Lucide React v0.563.0 |
| **Version Control** | Git / GitHub |
| **API Testing** | Postman |

---

### 2. Hardware Requirements
Minimum hardware specifications required for clinic staff and system maintenance.

*   **Computing Device**:
    *   **Type**: PC or Laptop (for clinic staff use)
    *   **Processor**: Intel Core i3 10th Gen or Ryzen 3 (Dual-core 2.4GHz+)
    *   **Memory (RAM)**: 8GB DDR4 (Minimum)
    *   **Storage**: 256GB SSD (for fast system performance)
*   **Connectivity**: 
    *   **Network**: Broadband Modem/Router with stable internet connection.
    *   **Mobile**: Active cellular connection for SMS reminder delivery.
*   **Peripherals**: 
    *   **Printer**: Laser or Inkjet printer for generating prescriptions and physical reports.
    *   **Display**: Standard 1080p monitor for dashboard visibility.

---

### 3. Core System Features (Per Thesis)
*   **AI-Integrated SMS Reminders**: Automated notifications for pet vaccinations and grooming schedules.
*   **Digital Patient Records**: Centralized storage for pet profiles and owner history.
*   **Electronic Prescription Management**: Linked prescriptions for better continuity of care.
*   **Inventory Tracking**: Real-time monitoring of clinical and grooming supplies.
